Student Grade Management System
project Description
The Student Grade Management System is a simple application used to calculate, store, and display student marks, total, average, and grades.
It helps teachers or students easily understand academic performance.

Objectives:
1. Enter student marks
2. Calculate total and average
3. Assign grades based on marks
4. Display student results clearly
     
Technologies Used:
1. Programming Language: Python 
2. Platform: Desktop
3. Tools: VS Code
     
Project Features:
1. Student name and roll number input
2. Marks entry for multiple subjects
3. Automatic grade calculation
4. Result display
     
Grade Criteria :
A : 90 – 100
B : 75 – 89
C : 60 – 74
D : 40 – 59
F : Below 40

How to Run the Project:
1. Download or clone the project
2. Open the file in your editor
3. Run the program
4. Enter student details and marks

Sample Output:
1. Student Name
2. Total Marks
3. Average
4.  Grade

Applications:
1. Schools
2. Colleges
3. Mini academic projects
4. Learning basic programming logic.
