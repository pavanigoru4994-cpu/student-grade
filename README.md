Student Grade Management System
project Description
The Student Grade Management System is a simple application used to calculate, store, and display student marks, total, average, and grades.
It helps teachers or students easily understand academic performance.

Objectives
     Enter student marks
     Calculate total and average
     Assign grades based on marks
     Display student results clearly
     
Technologies Used
     Programming Language: Python 
     Platform: Desktop
     Tools: VS Code
     
Project Features
     Student name and roll number input
     Marks entry for multiple subjects
     Automatic grade calculation
     Result display
     
Grade Criteria (Example)
A : 90 – 100
B : 75 – 89
C : 60 – 74
D : 40 – 59
F : Below 40

How to Run the Project
   Download or clone the project
   Open the file in your editor
   Run the program
   Enter student details and marks

Sample Output
   Student Name
   Total Marks
   Average
   Grade
